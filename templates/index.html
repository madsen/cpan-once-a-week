The CPAN “Once a Week, Every Week” Contest

In a blog entry, Mark Fowler resolved to release a Perl distribution to the CPAN each and every week. I decided to take him up on his challenge, and created this website to track our progress. It’s fairly obvious I’m a coder, not a graphic designer. Don’t complain about the design, fork it on GitHub and improve it! Pull requests welcome.

If you have a PAUSE ID, then you’re entered in the contest. All you have to do is upload a distribution to CPAN every week, and don’t skip any weeks. It can be a new module, or just a new version of an existing module. What’s the prize? The satisfaction of knowing you’re helping to make the CPAN the greatest collection of open-source code on the planet.

This page shows the top 10 chains in all categories. Click on the headings to see the full lists.
